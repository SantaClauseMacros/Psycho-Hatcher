Garden Mode Troubleshooting

Psycho Hatchers Garden Mode Troubleshooting Guide

This guide provides setup instructions and troubleshooting steps for the Garden Mode.

Setup Instructions

  1. Navigate to the Garden: Enter the garden area within the game.
  2. Position Yourself: It's important to sit near the Insta Plant Capsule and the Planter Pot to ensure proper functionality of the mode.
  3. Run the Mode: Start the automation process once you are correctly positioned.
  4. Display Settings: Make sure that Roblox is not running in full-screen mode, as this can cause issues with the automation.

Requirements for Garden Mode

To effectively use Garden Mode, you will need the following items:

  • Diamond Seed
  • Insta Plant Capsules

Common Issues

Incorrect positioning

Issue: User is not sitting in the correct position near the Insta Plant Capsule and Planter Pot.

Ask: "Are you sitting in the correct position near the Insta Plant Capsule and Planter Pot?"

Fix: Ensure you are positioned correctly near both the Insta Plant Capsule and Planter Pot. Misalignment can disrupt the automation.

Fullscreen mode issues

Issue: Roblox is running in fullscreen mode.

Ask: "Is Roblox running in fullscreen mode?"

Fix: Verify that Roblox is not in full-screen mode; switch to windowed mode if necessary.

Performance issues

Issue: System is experiencing lag or performance issues.

Ask: "Is your system experiencing lag or performance issues?"

Fix: System performance may affect the automation process. Ensure your system is running efficiently without excessive lag or FPS drops.

Missing required items

Issue: Missing Diamond Seed or Insta Plant Capsules.

Ask: "Do you have the Diamond Seed and Insta Plant Capsules?"

Fix: Verify that you have the required items in your inventory before starting the mode.

Scaling is not set to 100%

Issue: Display scaling is not set to 100%.

Ask: "Is your display scaling set to 100%?"

Fix: Adjust your PC's display scaling to 100% in Settings > Display > Scale and Layout.

Not using web browser version

Issue: Using the Roblox app instead of web browser version.

Ask: "Are you using the web browser version of Roblox?"

Fix: Close the Roblox app and use the web browser version instead.

Escalation Process

If all troubleshooting steps have been completed and the issue persists:

  • Contact any available staff member for assistance with technical issues.
  • For advanced troubleshooting, please escalate to a senior support team member.
