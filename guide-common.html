Common Issues FAQ

Display and System Settings

Display Scaling Issues

Issue: Display scaling not set to 100%

Solution: Set scaling to 100% in Settings > Display > Scale and Layout

Fullscreen vs. Windowed Mode

Issue: Game running in fullscreen mode

Solution: Switch to windowed mode via Esc > Settings > Display Mode

Browser Version Requirements

Issue: Using app version instead of browser version

Solution: Close the app and use the web browser version

Game-Specific Issues

Macro UI Problems

Issue: Macro interface overlapping with game

Solution: Adjust window size or reposition the macro interface

Hoverboard Issues

Issue: Shiny hoverboards interfering with macros

Solution: Switch to basic hoverboard or dismount completely

Performance Issues

System Lag

Issue: System performance affecting macro timing

Solution: Close background apps and reduce graphics settings

Browser Performance

Issue: Browser resource usage affecting performance

Solution: Close unused tabs and extensions
